# Abhi_site

पांडुरंग स्टिल चैनलिंक (जाळी) इंडस्ट्रीज — Marathi business website (Vite + React + Tailwind).

## GitHub Pages (live site)

1. On GitHub: **Settings → Pages → Build and deployment → Source: GitHub Actions**.
2. Push to `main`. The workflow **Deploy to GitHub Pages** builds and publishes `dist`.
3. After the workflow succeeds, open **Settings → Pages** for the site URL (e.g. `https://nileshgolande.github.io/Abhi_site/`).

## Local

```bash
npm install
npm run dev
npm run build
```
